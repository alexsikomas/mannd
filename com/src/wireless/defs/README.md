## Overview

This directory contains definitions for interacting with the `neli` crate for nl80211.

A majority of this code is from [neli-wifi](https://github.com/MaxVerevkin/neli-wifi), which itself is a fork of [nl80211](https://github.com/Eonm/nl80211).

## Purpose for Internal Version

A custom version is maintained here because the `neli-wifi` crate is not currently up-to-date with the latest API of the [neli](https://crates.io/crates/neli) crate.

Portions of this code have been modified to match the new `neli` API. However, a majority of the code remains unchanged.
